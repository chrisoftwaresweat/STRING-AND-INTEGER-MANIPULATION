# 🔤 String and Integer Manipulation

## 📌 Overview

This project is a simple Java-based program that performs various **string and integer manipulations**.
It focuses on strengthening fundamental programming concepts such as loops, conditionals, and data handling.

---

## 🚀 Features

* 🔄 Reverse strings and integers
* 🔢 Sort strings and integers
* 🔍 Check if a string or integer is a palindrome

---

## 💡 Purpose

This project was created to:

* Practice handling **mixed data types** (strings and integers)
* Strengthen problem-solving and logical thinking skills
* Serve as a reference for similar problems in the future
* Help other beginners understand how these operations work

---

## 💻 Examples

### 🔄 Reversing

**Input:** `009`
**Output:** `900`

**Input:** `dog`
**Output:** `god`

---

### 🔍 Palindrome Check

**Input:** `madam`
**Output:**
Reversed: `madam`
Result: ✅ Palindrome

**Input:** `900`
**Output:**
Reversed: `009`
Result: ❌ Not Palindrome

---

### 🔢 Sorting

**Input:** `231`
**Output:** `123`

**Input:** `cab`
**Output:** `abc`
