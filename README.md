## 📌 What this project does
Handles string and integer manipulation such as:
- reversing a string and integers
- sorting string and integers
- idnetifying if they're palindrome or not

## 💡 Why I made this
- To practice handling mixed data types (strings + integers)
- So I can look back and study these
- So anyone who's struggling in these certain problems gets to see how it's done

## ▶️ How to Run

1. Clone the repository
2. Open in any Java IDE (VS Code / IntelliJ)
3. Run the main file
4. Follow the menu prompts
